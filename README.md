# Installation

```
install.packages("devtools")
devtools::install_github("akirocode/akshiny")
library(akshiny)

```

or install a specific immutable version (specific tag)

```
install.packages("devtools")
devtools::install_github("akirocode/akshiny@0.8.0")
library(akshiny)

```
